# seastar tuturial

Code example in [seastar tutorial](https://github.com/scylladb/seastar/blob/master/doc/tutorial.md).

## Build

Seastar and C++20 compiler should be ready, see [build Seastar](./seastar-build.md)

Build and run:

```bash
mkdir build && cd build

cmake3 -G Ninja ..

ninja

./get-started
```
